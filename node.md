# Start Node

## Local Node

`$ ./scripts/pm2start.sh`

## Production Node

`$ ./scripts/pm2start.sh --env production`

# Purge chain

## Local Node

`$ ./scripts/purge-chain.sh`

## Production Node

`$ ./scripts/purge-chain.sh --env production`

# Stop Node

## Local Node

`$ ./scripts/pm2stop.sh`

## Production Node

`$ ./scripts/pm2stop.sh --env production`
